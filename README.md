# 📊 المحاسب البسيط — Telegram Mini App

تطبيق محاسبي احترافي مُصمَّم بشكل أساسي للهاتف المحمول (Mobile-First) ومتوافق مع Telegram Mini Apps، مع واجهة زجاجية (Glassmorphism) حديثة، تبويبات سفلية ثابتة، **ونظام تسجيل دخول/تسجيل آمن**.

---

## نظرة عامة على المشروع
- **الاسم**: المحاسب البسيط
- **الهدف**: إدارة الحسابات والمعاملات المالية بسهولة على الجوال مع خصوصية كاملة (التخزين محلي/Telegram Cloud).
- **المنصة**: Cloudflare Pages + Hono Framework

---

## ✨ المزايا الجديدة (آخر تحديث)

### 🆕 ما تم إضافته/تعديله في هذه النسخة
1. ✅ **تغيير اسم التطبيق** إلى **«المحاسب البسيط»** (في العنوان والشاشة الرئيسية والأيقونات)
2. ✅ **خطوط أجمل وأكثر احترافية**: استخدام **IBM Plex Sans Arabic** (للنصوص العربية) + **Rubik** (للأرقام) + **Almarai** كاحتياط
3. ✅ **أزرار أصغر وأنيقة**: تم تقليل الـ padding والارتفاع، نصوص أصغر، حواف منحنية أنيقة
4. ✅ **نقل الإجراءات السريعة لأعلى الصفحة الرئيسية** بالترتيب المطلوب: **معاملة → حساب → كشف → الحسابات**
5. ✅ **ميزة تعديل اسم الشركة (شركتي)**: زر قلم رصاص بجانب اسم الشركة في الهيدر — يفتح Modal للتعديل
6. ✅ **تنقل سلس** بين التبويبات بأنيميشن `cubic-bezier` ناعم وإعادة تشغيل الأنيميشن عند كل تبديل
7. ✅ **شاشة تسجيل دخول/تسجيل** كاملة:
   - تبويبان: **تسجيل الدخول** و **إنشاء حساب**
   - حقل اسم المستخدم + كلمة مرور (مع زر إظهار/إخفاء)
   - عند التسجيل: إدخال اسم الشركة + تأكيد كلمة المرور
   - **كلمات المرور مُشفّرة بـ SHA-256** (لا تُحفَظ كنص خام)
   - **بيانات كل مستخدم منفصلة تماماً** (مفاتيح تخزين خاصة بكل user)
   - حفظ الجلسة تلقائياً (auto-login عند الفتح)
   - زر تسجيل خروج في الهيدر

## 🗺️ مسارات الدخول (URLs)

| المسار | الوصف |
|---|---|
| `/` | الصفحة الرئيسية (تحويل إلى التطبيق) |
| `/static/app` | تطبيق المحاسب البسيط الكامل (SPA) |
| `/api/health` | فحص حالة الخادم (JSON) |

### تبويبات داخلية (عبر `switchTab()`)
- `dashboard` — لوحة التحكم (بها الإجراءات السريعة في الأعلى)
- `accounts` — إدارة الحسابات
- `transactions` — إضافة/عرض المعاملات
- `statement` — كشف الحساب + تصدير PDF

### دوال عالمية رئيسية (Frontend)
| الدالة | الوصف |
|---|---|
| `switchAuthTab('login'\|'register')` | تبديل بين تسجيل الدخول والتسجيل |
| `togglePassword(id, btn)` | إظهار/إخفاء كلمة المرور |
| `logout()` | تسجيل الخروج |
| `openAddAccount()` | فتح Modal إضافة حساب |
| `openEditCompany()` | فتح Modal تعديل اسم الشركة |
| `selectType('credit'\|'debit')` | اختيار نوع المعاملة |
| `deleteAccount(id)` | حذف حساب |
| `deleteTrans(id)` | حذف معاملة |
| `generateStatement()` | توليد كشف الحساب |
| `exportPDF()` | تصدير الكشف إلى PDF |

## 💾 معمارية البيانات (Data Architecture)

### 🔐 المستخدمون (Users)
**Key**: `simple_acc_users_v1` (localStorage مشترك)
```js
{
  "username1": {
    username: "username1",
    passwordHash: "<sha-256 hex>",
    createdAt: "ISO Date"
  },
  ...
}
```

### 🪪 الجلسة (Session)
**Key**: `simple_acc_session_v1`
```js
{ username: "...", loggedAt: "ISO Date" }
```

### ⚙️ إعدادات المستخدم (Per-User Settings)
**Key**: `simple_acc_settings__<username>`
```js
{ company_name: "شركتي" }
```

### 💼 بيانات التطبيق (Per-User Data)
**Key**: `simple_acc_data__<username>` + `simple_acc_meta__<username>`
نفس بنية النسخة السابقة:

#### Account
```js
{
  data_type: 'account',
  account_id: 'xxx',
  account_name: 'اسم الحساب',
  account_type: 'زبون' | 'مندوب' | 'شركة' | 'مورد' | 'أخرى',
  account_balance: 0,
  created_at: 'ISO Date',
  __backendId: 'xxx'
}
```

#### Transaction
```js
{
  data_type: 'transaction',
  transaction_id: 'xxx',
  transaction_account_id: 'ref-to-account',
  transaction_type: 'credit' | 'debit',
  transaction_amount: 100.00,
  transaction_currency: 'USD' | 'EUR' | 'TRY' | 'SYP',
  transaction_description: 'ملاحظة',
  transaction_date: 'YYYY-MM-DD',
  created_at: 'ISO Date',
  __backendId: 'xxx'
}
```

### ☁️ خدمات التخزين المستخدمة
- **localStorage** (افتراضي للمتصفحات)
- **Telegram Cloud Storage** (تلقائي عند فتح التطبيق داخل Telegram)
- **مزامنة تلقائية** بعد كل عملية + زر مزامنة يدوي

> 🚫 لم يتم استخدام Cloudflare D1/KV/R2 لأن البيانات شخصية وحساسة وتُحفَظ على جهاز المستخدم.

## 🧭 دليل المستخدم السريع

### 1️⃣ أول مرة تفتح التطبيق
- ستظهر **شاشة الترحيب** بـ تسجيل الدخول
- اضغط **«إنشاء حساب»** ثم أدخل: اسم الشركة، اسم مستخدم، كلمة مرور (4 أحرف على الأقل)، وأكّد كلمة المرور
- اضغط **«إنشاء الحساب»** لتدخل مباشرة

### 2️⃣ الاستخدام اليومي
- في الأعلى: **الإجراءات السريعة** — معاملة، حساب، كشف، الحسابات
- اضغط على القلم 🖊️ بجانب اسم شركتك لتعديله
- استخدم التبويبات السفلية للتنقل بين: الرئيسية / الحسابات / المعاملات / الكشف
- اضغط **مزامنة** لمزامنة بياناتك يدوياً (تتم تلقائياً أيضاً)

### 3️⃣ تسجيل الخروج
- اضغط زر **🔓** في أعلى الشاشة بجانب زر المزامنة

## 🛠️ التقنيات المستخدمة

- **Backend**: Hono Framework (Cloudflare Pages/Workers)
- **Frontend**: HTML5 + Tailwind CSS (CDN) + Vanilla JavaScript
- **خطوط**: IBM Plex Sans Arabic + Rubik + Almarai (Google Fonts)
- **أيقونات**: Lucide Icons
- **Telegram WebApp SDK** لتكامل Mini App
- **html2pdf.js** لتصدير PDF
- **Web Crypto API** لتشفير كلمات المرور (SHA-256)

## 🚀 النشر والتشغيل (Deployment)

### تشغيل محلي في الـ Sandbox
```bash
cd /home/user/webapp
npm run build
pm2 start ecosystem.config.cjs
curl http://localhost:3000
```

### نشر على Cloudflare Pages
عبر تبويب **Publish** في الواجهة، أو:
```bash
cd /home/user/webapp
npm run deploy
```
